Motion Graphics Video

Some Context

This video was created for a upper division film course.Â I had complete freedom to create any type of video to represent myself. As such I chose to highlight my skill in using motion graphics. This required learning many new motion graphic techniques and transitions from dozens of online tutorials as well several iterations of refinement.

ROLES: Editor, Visual Effects Editor.

My Process.

Over the course of two weeks this video has changed dramatically. From transitions, to the various animations and many of the effects used throughout this video, not one effect is the same as the initial draft. After dozens of iterations, the timing, complexity and typography of the video has been refined into the current iteration. One example of one effect upon being iterated is a set of text changing from a simple z rotation to a 3D animation sequence as shown below. This workflow presented an opportunity to learn about syncing motion graphics animation to music and other effects.

Learning.

This project afforded an opportunity to learn several new animation effects including using 3D text, learning basic kinetic typography and some stylish transitions. I was even able to learn how to use adobe after effects suite of 3D effects and rendering, of which I have never used prior to creating this video. This process requires synthesizing materials found through online tutorials, techniques learned through my university curriculum and my own intuition.

Refinement.

Creating this video required implementing countless small refinements and tweaks to the timing of keyframes for animations and transitions. Through a constant cycle of reiteration I was able to spot issues and strive to amend them. This process required a stern dedication in order to implement these improvements, especially after 20+ renders. From this project I have learned a tremendous amount about keyframe animation and motion graphic creation. Additionally I have found new value in a iterative design process to continuously create prototypes until reaching a polished final product.
